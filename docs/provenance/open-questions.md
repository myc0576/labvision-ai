# Provenance Open Questions

The MIT public core release does not include the legacy-dependent modules listed below. These questions must be answered before restoring those features:

1. Confirm ownership and redistribution status for the legacy two-dimensional LIF/MIE source responsibilities.
2. Confirm ownership and redistribution status for research-workbench layout/export responsibilities.
3. Decide whether any real experimental images may be redistributed; current examples use synthetic arrays only.
4. Add an approved release decision record before any package-index upload or external distribution beyond the GitHub core repository.

Allowed now: public GitHub browsing, local editable installs, examples, tests, local wheel/sdist builds, and clean-room core development.
